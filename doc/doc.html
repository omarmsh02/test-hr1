WorkWise

HR Management System

Executive Summary

WorkWise is a robust web-based Human Resources Management System (HRMS) designed to centralize and automate HR processes in small to mid-sized companies. Developed using Laravel, Blade templates, HTML/CSS, and MySQL, WorkWise enables HR personnel, managers, and employees to interact efficiently across departments. The platform reduces administrative overhead and enhances transparency and accountability.

Technology Stack

  • Backend Framework: Laravel (PHP)
  • Frontend: Blade templating engine, HTML, CSS, JavaScript
  • Database: MySQL
  • Authentication: Laravel's native authentication with middleware-based role access
  • Access Control: Role-based middleware (AdminMiddleware, ManagerMiddleware, EmployeeMiddleware)

Role-Based Architecture

Users access the system from a unified login interface. After login, Laravel middleware identifies the user's role and redirects them to the correct dashboard:
